# Cross-Platform Installation

Python scripts that work on any OS (Linux, macOS, Windows).

## Requirements

- Python 3.10+
- pip
- git (for source and claude-code)

## Installation Methods

| Method | Script | Description |
|--------|--------|-------------|
| [pip](./pip/) | `python install.py` | Standard installation |
| [source](./source/) | `python install.py` | Development / latest |
| [claude-code](./claude-code/) | `python install.py [project]` | Full Claude Code integration |

## Usage

```bash
python install.py
```

Works identically on Linux, macOS, and Windows.
