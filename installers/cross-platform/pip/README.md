# pip Installation (Cross-Platform)

## Quick Start

```bash
python install.py
```

Or directly:
```bash
pip install vibesrails
```

## After Installation

```bash
cd your-project
vibesrails --setup
```

## Troubleshooting

### Command not found

```bash
python -m vibesrails --version
```

### Permission denied

```bash
pip install --user vibesrails
```
