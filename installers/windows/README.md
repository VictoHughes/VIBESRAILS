# Windows Installation

Batch scripts for Windows.

## Requirements

- Python 3.10+ (from python.org)
- git (from git-scm.com)

## Installation Methods

| Method | Script | Description |
|--------|--------|-------------|
| [pip](./pip/) | `install.bat` | Standard installation |
| [source](./source/) | `install.bat` | Development / latest |
| [claude-code](./claude-code/) | `install.bat [project]` | Full Claude Code integration |

## Usage

Double-click the `.bat` file or run from Command Prompt:

```cmd
install.bat
```
