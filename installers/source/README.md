# Source Installation

Install from source for development or latest features.

## Quick Install

```bash
./install.sh
```

This will:
1. Clone the repository to `~/.vibesrails`
2. Install in development mode (`pip install -e .`)

## Manual Installation

```bash
git clone https://github.com/VictoHughes/VIBESRAILS.git
cd VIBESRAILS
pip install -e .
```

## After Installation

```bash
cd your-project
vibesrails --setup
```
