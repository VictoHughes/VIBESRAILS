# Unix/Mac Installation

Shell scripts for Linux and macOS.

## Requirements

- Python 3.10+
- bash
- git (for source and claude-code)

## Installation Methods

| Method | Script | Description |
|--------|--------|-------------|
| [pip](./pip/) | `./install.sh` | Standard installation |
| [source](./source/) | `./install.sh` | Development / latest |
| [claude-code](./claude-code/) | `./install.sh [project]` | Full Claude Code integration |

## Usage

```bash
# Make executable (if needed)
chmod +x install.sh

# Run
./install.sh
```
