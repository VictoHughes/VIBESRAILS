# v2.5.0 Feature Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add 6 features to VibesRails: bandit_scan, coverage_check, impact_check, mcp_audit, project_type_detect, OpenCode hooks adapter.

**Architecture:** Each feature follows established patterns -- adapter pattern for external tools (Bandit), guard pattern for AST analysis (impact_check), context signals for detection (project_type), JSON parsing for readers (coverage, mcp_audit). OpenCode adapter generates a JS plugin that wraps existing Python hooks.

**Tech Stack:** Python 3.10+, stdlib ast, subprocess, json. Optional deps: bandit (SAST). Zero new required deps.

**Tasks:** 10 tasks, ~70 steps. Estimated ~60 new tests.
